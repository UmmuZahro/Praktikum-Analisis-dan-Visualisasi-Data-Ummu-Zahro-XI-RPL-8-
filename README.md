Analisis Performa Penjualan E-Commerce

Dataset
Brazilian E-Commerce Public Dataset by Olist

Tujuan
Menganalisis performa penjualan e-commerce berdasarkan tren penjualan, produk terlaris, dan segmentasi pelanggan.

Analisis yang Dilakukan
Tren Penjualan Bulanan
Produk Terlaris
Heatmap Korelasi
RFM Analysis
Produk Underperformer
